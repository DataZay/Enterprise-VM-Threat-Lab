# 📊 Ubuntu Wazuh SIEM Server

## 🔧 VM Specs
- Ubuntu 22.04 Server
- Hostname: `wazuh-server`
- IP: `10.0.0.10`
- RAM: 4096 MB

## 🏗️ Installation Stack
- Wazuh Manager
- Wazuh Dashboard (Kibana)
- Elasticsearch

## 🔐 Dashboard Access
- URL: `https://localhost`
- Login: `admin / <password>`

## 🧭 Log Flow
Wazuh Agent → Wazuh Manager → Elasticsearch → Wazuh Dashboard

## 🛡️ Sample Detection Rules
- Unauthorized login
- Brute force SSH
- Suspicious PowerShell
- Netcat reverse shells
